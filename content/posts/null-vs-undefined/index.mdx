null과 undefined는 둘 다 없음을 나타내는 값이지만 미세한 차이가 있습니다.

## undefined

값을 할상하지 않은 변수

메서드와 선언에서 변수가 할당받지 못한 경우

함수가 값을 return 하지 않을 때

즉, 값이 정해지지 않은 상황에서 기본적으로 부여되는 값입니다.

## null

null은 undefined와는 다르게 의도적으로 값을 비운 값입니다.

---

typeof undefined는 undefined로 출력되지만

typeof null을 출력하면 object입니다.
