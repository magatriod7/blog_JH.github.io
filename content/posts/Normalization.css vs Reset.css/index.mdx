### Reset.css

각 브라우저마다 설정된 기본 style이 다르게 설정되어 있다. 심지어 익스플로어(지금은 죽었지만)는 각 버전마다 그 설정이 달랐다. 이를 방지하기 위해 모든 스타일 값을 0 혹은 default로 reset 시켜 작업을 할 수 있도록 해주는 것이 reset.css를 칭한다.


### Normalize.css

normalize는 브라우저의 기존 스타일을 전부 리셋시키는 것이 아닌 이를 일부 유지하고 이용하는 방법이다. 브라우저가 가지고 있는 버그를 수정하는 목적도 있다.

Reset 단점 : 모든 스타일링을 초기화하는 것이기 때문에 시간이 오래걸린다. 불필요한 재선언을 많이 필요한다.
Reset 장점 : 지속적인 업데이트가 필요 없다.

normalize 단점 : 브라우저의 업데이트로 인하여 스타일이 변경될 수 있어 지속적인 업데이트가 필요하다.
normalize 장점 : 디버깅이 비교적 쉽다. 초기화를 원하는 부분을 적당히 설정하여 원하는 방향으로 normalize가 가능하다.

결론 : 원하는 것을 쓰면 된다고 생각한다.
