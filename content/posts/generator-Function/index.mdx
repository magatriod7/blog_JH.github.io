일반적인 함수는 하나의 값을 반환합니다. 하지만 제네레이터를 사용하면
여러개의 값을 반환이 가능합니다.

```
function* generatorExample() {
  yield 1;
  yield 2;
  return 3;
}

// '제너레이터 함수'는 '제너레이터 객체'를 생성합니다.
let generator = generatorExample();
console.log(generator); // [object Generator]
```

생성된 제네레이터 함수에 next() 메소드를 호출하게 되면 다음 yield가 나올 때까지 반복하게 됩니다.
이 후 next는 value(yield의 산출 값)과 done(함수 코드의 끝남 유무)
를 호출합니다.

```

// *을 붙이면 제네레이터 함수가 됩니다
// function* f(...), function *f(...) 둘 다 가능합니다. 전자가 더 선호됩니다.
function* generatorExample() {
  yield "테스트";
  yield 2;
  return 3;
}

// '제너레이터 함수'는 '제너레이터 객체'를 생성합니다.
let generator = generatorExample();
let next = generator.next();
console.log(next);// "value":"테스트", "done" : false
next = generator.next();
console.log(next);// "value":2, "done" : false
next = generator.next();
console.log(next); // "value":3, "done" : true
```

제네레이터 함수는 이터러블입니다. 그래서 다음과 같은 것도 가능합니다.

```
function* generateSequence() {
  yield 1;
  yield 2;
  yield 3;
}

let generator = generateSequence();

for(let value of generator) {
    console.log(value); // 1, 2, 3이 출력됨
}

let generator2 = [0, ...generateSequence()]

console.log(generator2, "sequence")//[0,1,2,3] "sequence"
```

제네레이터를 이용해서 동기식 코드가 작성이 가능해집니다. 사용할 수 있는 대표적인 예시로는 redux-saga 등이 있습니다.
