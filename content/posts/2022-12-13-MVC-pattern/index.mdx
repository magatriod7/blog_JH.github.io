MVC 패턴은 Model View Controller의 줄임말로

model : data

view : UI

Controller : logic

를 담당하는 컴포넌트로 나눠서 컴포넌트를 관리하는 것입니다.

MVC 패턴은 1978년에 Trygve Reenskaug가 발견했습니다.
